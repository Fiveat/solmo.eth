Where are these NFTs?
different networks




NFT stands for "non-fungible token." At a basic level, an NFT is a digital asset that links ownership to unique physical or digital items, such as works of art, real estate, music, or videos. NFTs can be considered modern-day collectibles.



My NFTs are been uploaded on Ethereum Network (this own website data) and my own Licenses and Certificates are been uploaded on Solana Network. All NFTs can be uploaded on different networks. Each network, provides a features to developers. To deploy code on Ethereum, you need Solidity. To deploy code on Solana, you need TypeScript, JavaScript and other common lenguajes.
